Expand description

Variable names for pathconf

Nix uses the same naming convention for these variables as the getconf(1) utility. That is, PathconfVar variables have the same name as the abstract variables shown in the pathconf(2) man page. Usually, itโs the same as the C variable name without the leading _PC_.

POSIX 1003.1-2008 standardizes all of these variables, but some OSes choose not to implement variables that cannot change at runtime.
