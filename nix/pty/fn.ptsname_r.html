Get the name of the slave pseudoterminal (see ptsname(3))

ptsname_r() returns the name of the slave pseudoterminal device corresponding to the master referred to by fd. This is the threadsafe version of ptsname(), but it is not part of the POSIX standard and is instead a Linux-specific extension.

This value is useful for opening the slave ptty once the master has already been opened with posix_openpt().
